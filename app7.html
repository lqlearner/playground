Financial Calendar Converter

How to Use

  1. Open the Economic Calendar in a new tab: Click here to open the calendar.
  2. In that new tab, select your desired date range and filters.
  3. Right-click anywhere on that page and select "View Page Source".
  4. A new tab will open with the HTML code. Press Ctrl+A (or Cmd+A) to select all, then Ctrl+C (or Cmd+C) to copy.
  5. Return to this page and paste the code (Ctrl+V or Cmd+V) into the text box below.
  6. Click the "Convert Calendar" button.
