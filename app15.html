提示：Chrome 在 Windows 上会直接把网页通知显示到系统通知栏（右下角）。第一次需要用户授权（HTTPS 或 localhost）。

如何接收 Windows 系统通知：
  • 本网页使用标准的 Notification API 发送通知。
  • 在 Windows 上的 Chrome 浏览器中，这些通知会自动显示为 原生系统通知（桌面右下角弹窗），并被记录在 Windows 的“通知中心”里。
  • 首次使用时，浏览器会请求授权。请点击上方的 Enable notifications 按钮，然后在浏览器弹窗中选择“允许”。
  • 此功能需要安全的网络环境 (HTTPS) 或本地开发环境 (localhost)。
  • 注册 Service Worker 后，通知上还可以带“Snooze”等快捷操作按钮。
